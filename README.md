# customer-behaviour-analysis
Analysis of customer shopping behavior using Python and SQL. Includes data cleaning, EDA, visualizations, and actionable business insights.
📊 Customer Behavior Analysis

This project explores customer shopping behavior to identify trends, patterns, and insights that help understand purchasing habits and improve business decision-making. The analysis includes customer segmentation, spending behavior, seasonal trends, and product preferences.

📁 Dataset Overview

The dataset includes information such as:

Customer ID
Gender
Age
Category
Item Purchased
Purchase Amount (USD)
Discount Applied
Previous Purchases
Season
Review Rating
🎯 Project Objectives
Analyze customer spending and purchasing patterns
Identify high-revenue customer segments
Discover seasonal sales trends
Understand the impact of discounts
Find top-selling items in each category
Build insights using SQL and Python
🛠️ Tech Stack Used
Python (Pandas, Matplotlib, Seaborn)
SQL (PostgreSQL)
Jupyter Notebook
SQLAlchemy (for database connectivity)
🔍 Key Analysis Performed
✔ Data Cleaning
Removed inconsistencies
Converted data types
Handled missing values
✔ Exploratory Data Analysis
Revenue by demographic groups
Most purchased items by category
Seasonal purchase patterns
Discount usage and impact
Customer segmentation (New, Returning, Loyal)
✔ SQL Insights

Analysis includes:

Revenue by gender
Top 3 items in each category
High-spending customers
Discount percentage per item
Customer segmentation counts
📊 Visualizations

The notebook includes visual insights such as:

Revenue by Season
Category-wise sales
Customer segment distribution
Discount vs Purchase Amount
Rating vs Spending
